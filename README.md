📊 AtliQ Hardware Sales Insights Dashboard (Power BI)
🚀 Problem Statement

AtliQ Hardware is a company that supplies computer hardware and peripherals across multiple regions in India. The Sales Director faced challenges in understanding business performance due to:

Declining sales trends
Lack of clear, real-time insights
Dependence on Excel reports
Difficulty in interpreting raw numerical data

Frequent manual reporting from regional managers led to inefficiencies and delayed decision-making.

💡 Solution

To overcome these challenges, a Power BI Dashboard was developed to transform raw data into actionable insights through interactive visualizations, enabling data-driven decision-making.

🎯 Project Planning (AIMS Grid)
Aim: Provide clear visibility into sales performance and identify key business issues
Stakeholders: Sales Director, Regional Managers, Business Analysts
End Result: Interactive dashboard with KPIs and insights
Success Criteria:
Faster decision-making
Improved visibility into revenue & profit trends
Identification of high/low performing markets & products
🛠️ Tech Stack
Database: MySQL
Visualization Tool: Power BI
Data Processing: Power Query (ETL)
Language: DAX
⚙️ Steps Followed
Understood business requirements using AIMS Grid
Extracted data from MySQL database
Performed data cleaning using Power Query
Applied ETL process (Extract, Transform, Load)
Created DAX measures for KPIs and analysis
Standardized currency across transactions
Validated data for accuracy
Built data model and relationships
Designed interactive dashboards and visuals
🔧 Major Changes / Customizations
Resolved (blank) product issue by:
Rebuilding the product table with additional entries (Prod280–Prod339)
Assigning product types (Own Brand / Distribution)
Enhanced sales data by:
Merging transaction tables
Adding fields like profit margin and cost price
