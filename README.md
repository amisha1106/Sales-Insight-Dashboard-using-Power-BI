# 📊 AtliQ Hardware Sales Insights Dashboard (Power BI)

## 🚀 Problem Statement
AtliQ Hardware is a company that supplies computer hardware and peripherals across multiple regions in India. The Sales Director faced challenges in understanding business performance due to:

- Declining sales trends  
- Lack of real-time insights  
- Heavy dependence on Excel reports  
- Difficulty interpreting raw numerical data  

Frequent manual reporting from regional managers led to inefficiencies and delayed decision-making.

---

## 💡 Solution
Developed an interactive **Power BI Dashboard** to convert raw data into meaningful visual insights, enabling faster and data-driven decision-making.

---

## 🎯 Project Planning (AIMS Grid)

- **Aim**: Provide clear visibility into sales performance and identify business issues  
- **Stakeholders**: Sales Director, Regional Managers, Analysts  
- **End Result**: Interactive dashboard with KPIs and insights  
- **Success Criteria**:
  - Faster decision-making  
  - Improved revenue & profit visibility  
  - Identification of high/low performing markets & products  

---

## 🛠️ Tech Stack

- **Database**: MySQL  
- **Visualization**: Power BI  
- **Data Processing**: Power Query (ETL)  
- **Language**: DAX  

---

## ⚙️ Steps Followed

1. Understood business requirements using AIMS Grid  
2. Extracted data from MySQL database  
3. Performed data cleaning in Power Query  
4. Applied ETL process (Extract, Transform, Load)  
5. Created DAX measures for KPIs  
6. Standardized currency across transactions  
7. Performed data validation  
8. Built data model and relationships  
9. Designed interactive dashboards  

---
